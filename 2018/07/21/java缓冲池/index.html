自动装箱、拆箱

基本类型都有其对应的包装类型，基本类型与其对应的包装类型间的转换是靠自动装箱 与拆箱完成的

  • 基本类型 / 包装类
  • boolean / Boolean
  • char / Character
  • int / Integer
  • byte / Byte
  • short / Short
  • long / Long
  • float / Float
  • double / Double
